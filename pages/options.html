Options

Scrolling Behaviour

auto
The scrolling box scrolls instantly.

smooth
The scrolling box scrolls in a smooth fashion using a user-agent-defined timing function over a user-agent-defined period of time. User agents should follow platform conventions, if any.

Default: smooth


px

Default: 200

px

Default: 200

Available Keys

Pressing this key, the extension enters into scroll up node and does its job. Default: j

Pressing this key, the extension enters into scroll down node and does its job. Default: k

Pressing this key, the extension enters into click node and does its job. Job includes searching visible tags valid to be clicked and generates another graph to be appended to this node. Also starts to render the overlay of paths to click the generated nodes. Default: f

Hint keys. Keys to traverse nodes to select a element to click. Default: h, g, c

Tag Selectors

Any custom css selector can be used. Default: button and a.
